# trie-nim

## How to run

```bash
nim r trie.nim
```

## How to run tests
```bash
nim r tests/test_trie.nim
```

## What is?
- its a prefix tree
- tree data structure for efficient string storage O(n)

## Use Cases
- Autocomplete
- URL Routing(longest prefix match)
- prefix search system
